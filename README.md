# Timmy's Portfolio Website

個人作品集網站，展示我的技能、作品和經歷。

## 功能特點

- 🎨 科技感單頁式網站設計
- 🌓 深色/淺色模式
- 📱 RWD響應式設計
- ✨ 動畫效果
- 🖼️ 作品集展示
- 📝 經歷時間軸

## 技術棧

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts

## 如何運行

1. 使用 VS Code 的 Live Server 插件
2. 或使用 Python 的簡易服務器：
```bash
python -m http.server
```
3. 在瀏覽器中訪問 `http://localhost:8000`
